# Sistema de Gestión de Somnografías

Aplicación web desarrollada como proyecto final para la materia Técnicas de Programación.  
El sistema permite gestionar pacientes, registrar somnografías asociadas a cada paciente y programar estudios en una agenda.

## Objetivo

El objetivo de la aplicación es facilitar la organización de pacientes y estudios de somnografía en un centro especializado, permitiendo registrar información clínica básica, consultar estudios, administrar estados y visualizar estudios programados.

## Tecnologías Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap
- JavaScript
- HTML
- CSS
- Maven

## Funcionalidades Principales

- Página de inicio con información general sobre somnografías.
- Registro de pacientes.
- Listado y búsqueda de pacientes.
- Edición y eliminación de pacientes.
- Vista de detalle del paciente.
- Registro de somnografías asociadas a pacientes.
- Listado de somnografías.
- Vista de detalle de somnografías.
- Edición y eliminación de somnografías.
- Agenda de estudios programados.
- Estados visuales para las somnografías.
- Validaciones de formularios.
- Mensajes de éxito y error.
- Confirmaciones antes de eliminar registros.
